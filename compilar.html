Como compilar el juego - 2.0 y anteriores

Antes de empezar, cabe aclarar que se necesitan una serie de requisitos para compilar:

  • Máquina virtual con Windows XP
  • Una versión de PiX Pang que contenga la carpeta src
  • Una versión de Fénix anterior o igual a la 0.84 (Acá la descarga)
  • Un poco de sentido común y conocimiento básico

Una vez todo eso, se debe extraer el archivo donde está Fénix y buscar los archivos FXI.exe, SDL.dll, SDL.mixer.dll, zlib.dll y FXC.exe. Finalmente, los tenemos que pasar a la carpeta src.

Ahora, para compilar, se tendrá que crear un archivo bat con el siguiente código:

echo off
cls
echo Compilando pixpang.prg
fxc -g pixpang.prg
echo pixpang.prg compilado
pause

Dependiendo de la versión, puede llegar a fallar por la falta de librerías. En tal caso, solo hay que buscar la librería necesaria en la carpeta dll de Fénix. Si todo estuvo bien, se generará un archivo llamado pixpang.dcb.

Finalmente, para ejecutarlo, se crea otro bat con el siguiente código:

echo off
cls
echo Abriendo pixpang.dcb
fxi pixpang.dcb
